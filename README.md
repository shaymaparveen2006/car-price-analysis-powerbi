# # Car Price Analysis Dashboard

This project was created as part of the AICW Fellowship Program.

## Tools Used
- Microsoft Excel
- Power BI
- ChatGPT
- Kaggle Dataset

## Project Description
The project analyzes car price data and builds an interactive Power BI dashboard to understand pricing trends, brand distribution, mileage impact, and state-wise vehicle data.

## Dashboard Features
- Average, Minimum and Maximum Selling Price
- Vehicle Production by Brand
- Odometer vs Selling Price Analysis
- Geographic Distribution of Vehicles
- Market Price vs MMR Comparison
